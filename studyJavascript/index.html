1. 모든 웹 브라우저에서 작동한다.
2. 실행 결과를 즉시 확인할 수 있다.
3. 풀스택 웹 개발 뿐만 아니라 다양한 용도의 프로그램을 만들 수 있다.
-> HTTP 서버를 만들어, 브라우저로 요청이 들어왔을 때 "Hello, Node.js!"라는 메시지를 응답하는 코드 예시입니다.


                    // 1. http 모듈을 가져옵니다.
                    const http = require('http');
                    
                    // 2. 서버를 생성합니다.
                    const server = http.createServer((req, res) => {
                    // 3. 응답 헤더 설정 (200: 성공 상태, Content-Type: 텍스트)
                    res.writeHead(200, { 'Content-Type': 'text/plain' });
                    
                    // 4. 브라우저로 "Hello, Node.js!" 메시지를 보냅니다.
                    res.end('Hello, Node.js!\n');
                    });
                    
                    // 5. 서버가 3000번 포트에서 요청 대기 중임을 알립니다.
                    server.listen(3000, () => {
                    console.log('Server running at http://localhost:3000/');
                    });
