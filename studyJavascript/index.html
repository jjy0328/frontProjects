-> HTTP 서버를 만들어, 브라우저로 요청이 들어왔을 때 "Hello, Node.js!"라는 메시지를 응답하는 코드 예시입니다.


                    // 1. http 모듈을 가져옵니다.
                    const http = require('http');
                    
                    // 2. 서버를 생성합니다.
                    const server = http.createServer((req, res) => {
                    // 3. 응답 헤더 설정 (200: 성공 상태, Content-Type: 텍스트)
                    res.writeHead(200, { 'Content-Type': 'text/plain' });
                    
                    // 4. 브라우저로 "Hello, Node.js!" 메시지를 보냅니다.
                    res.end('Hello, Node.js!\n');
                    });
                    
                    // 5. 서버가 3000번 포트에서 요청 대기 중임을 알립니다.
                    server.listen(3000, () => {
                    console.log('Server running at http://localhost:3000/');
                    });

Scope : 변수를 선언하고 변수가 적용되는 범위로, 변수가 어디까지 유효한지 범위를 가리키는 영역입니다.

Local Variable : 한 함수 안에서 사용할 수 있는 변수

Global Variable : 스크립트 소스 전체에서 사용할 수 있는 변수

1. var
var는 자바스크립트에서 오래된 변수 선언 방식으로, 전역 혹은 함수 범위로 지정됩니다. 

                // Global Scope(전역범위) 예시
                var globalVar = "I am a global variable";
                function checkGlobalVar() {
                console.log(globalVar); // 전역 변수에 접근 가능
                }

                checkGlobalVar(); // "I am a global variable"
                console.log(globalVar); // "I am a global variable"

                // Function Scope (함수 범위)예시
                function testVar() {
                var functionScopedVar = "I am inside a function";
                console.log(functionScopedVar); // 함수 내부에서는 접근 가능
                }

                testVar(); // "I am inside a function"
                console.log(functionScopedVar); // ReferenceError: functionScopedVar is not defined
